# Banggood Clone

Bangood is an E-commerce website.

## Installation

Use the package manager [npm](https://docs.npmjs.com/cli/v6/commands/npm-install) to install the node module files.

```bash
npm install
```

## Usage

```python
cd banggood-clone
npm start
```

## Deployment Link
